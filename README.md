# DigitalSafe Spring Boot API

โปรเจกต์ตัวอย่าง Spring Boot สำหรับจำลอง Digital Safe

## Features
- สร้างตู้เซฟพร้อมชื่อเจ้าของ
- ฝากเงิน (deposit)
- ดูยอดเงินปัจจุบัน
- ดูชื่อเจ้าของตู้เซฟ

## API Endpoints
- GET /safe/owner
- GET /safe/balance
- POST /safe/deposit?amount=NUMBER

## Tech Stack
- Java
- Spring Boot
- Maven
